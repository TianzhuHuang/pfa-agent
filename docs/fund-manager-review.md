# PFA 产品评审 — 以基金经理视角

> 评审人：模拟角色 — 管理 ¥130 万、11 只标的、4 个账户的主观多头基金经理
> 投资风格：价值投资为主，关注基本面和渠道数据，适度关注技术面
> 日常工作：7:30 晨会 → 9:30 盘中跟踪 → 15:00 复盘 → 晚间研究

---

## 一、最关键的缺失：我打开后不知道"今天该干什么"

### 问题
当前 Portfolio Hub 展示的是**静态持仓列表**。但作为基金经理，我打开工具的第一眼需要的是：

**"今天我的持仓有什么异常？需要我做什么决策？"**

### 建议

**在持仓表格上方增加「今日异常」卡片**（Exception-Driven 设计，Bloomberg 核心理念）：

```
┌─────────────────────────────────────────────────────┐
│ 🔴 中海油 -3.2% 触发预警    🟡 茅台 传出厂价调整传闻  │
│ 📊 白酒板块今日回调 -1.5%   💰 人民币升破 6.84        │
└─────────────────────────────────────────────────────┘
```

这一行就能让我在 3 秒内判断今天要不要操作。

## 二、持仓表格缺失关键列

### 当前列
Symbol, Name, Price, Cost, Qty, Value, Return, %

### 缺失但必要的列

| 列 | 理由 |
|---|---|
| **今日涨跌额** | 我需要知道今天具体赚/亏了多少，不只是总盈亏 |
| **仓位占比 %** | 仓位管理是基金经理最核心的工作 |
| **持仓天数** | 判断短线还是长线，辅助决策 |
| **行业** | 一眼看出行业集中度 |

### 建议表头

```
Symbol  Name    Sector   Price   Chg%   Cost    Qty    Value    Weight%  P&L     Today

```

## 三、缺少仓位分析视图

作为基金经理，我每天需要看三张图：

### 1. 仓位占比饼图 (Position Sizing)
- 按个股占比：茅台 43%、中海油 11%、五粮液 5%...
- **红线规则**：单只超过 30% 要高亮警告

### 2. 行业/市场分布
- 按行业：白酒 48%、石油 15%、电力 5%...
- 按市场：A 股 60%、港股 35%、美股 5%

### 3. 盈亏贡献瀑布图
- 今日盈亏中，每只股票贡献了多少
- 正的在上面（红），负的在下面（绿）

### 问题
当前 AI Expert 只在我主动提问时才回答。但好的投研助理应该**主动发现问题**。

### 建议：AI 主动分析（每天自动）

**盘前（8:00）**：
```
🤖 早上好。基于你的持仓，今日需要关注：

1. 中海油：昨夜布伦特原油跌 2.1%，可能影响今日开盘

3. 你的白酒仓位 48%，超过分散化建议线 30%，建议关注

需要我展开分析某一条吗？

```
🤖 今日复盘：

总收益：-¥8,200 (-0.6%)

今日下跌主因油价短期回调，基本面未变。

是否需要调整仓位？
```

## 五、文案问题（不专业的地方）

| 当前文案 | 问题 | 建议修改 |

| "Total Portfolio" | 混合中英文 | 统一为中文"总资产"或全英 |

```

当前缺失：PE、PB、市值、股息率、52 周高低 — 这些是基金经理做决策的**基础数据**。

### 中栏：不应该只有新闻
应该有 Tab 切换：
- **新闻** (当前已有)
- **公告** (年报/季报/减持公告 — 最重要的硬数据)
- **研报** (券商评级变动)
- **讨论** (雪球/股吧情绪)

### 右栏：AI 分析不应该是空白聊天
应该预加载一段**自动生成的个股研报摘要**：
```
🤖 贵州茅台 快速研判

基本面：PE 25.3x 处于近5年 40% 分位，估值中性。
渠道：经销商反馈缺货，批价企稳。
风险：出厂价上调传闻被辟谣，短期可能压制股价。
你的备忘："看好渠道改革长期逻辑"— 当前基本面未证伪。

综合评价：维持持有 ★★★★☆
```

## 七、晨报页面缺实际投资决策支持

当前晨报是"新闻摘要"，但基金经理需要的是**决策支持**。

### 建议增加

1. **持仓标的评级变动**：今天哪家券商上调/下调了我持仓标的的评级？
2. **关联标的异动**：我没持有但应该关注的标的（如茅台的竞品五粮液大涨/大跌）
3. **宏观日历**：今天有什么重要数据公布？（CPI、PMI、美联储会议）
4. **操作建议汇总**：基于所有信息，今天建议"持有不动"还是"关注某只标的"

## 八、实际操作中的交互痛点

1. **没有"快速调仓"入口**：我在持仓表格里看到中海油跌了想减仓，需要打字告诉 AI，没有一键"买入/卖出"按钮
2. **没有行情刷新按钮**：不知道价格是几点的，需要手动刷新吗？应该显示"最后更新: 14:35"
3. **没有收盘/盘中状态指示**：A 股 15:00 收盘后，价格旁应标"收"
4. **没有排序**：持仓表格无法按涨跌幅、市值、盈亏排序
